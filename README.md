# Cascade EV

Static website package for a Tesla/EV rental, future sales, and future service brand.

## Included pages
- Home
- Rentals
- Future Sales
- Service
- Contact

## Before launch
1. Replace `https://cascadeev.com/` canonicals if using a different domain.
2. Replace Turo placeholder links with your exact listing URLs.
3. Replace email and phone values in `contact/index.html`.
4. Add real photos if desired.
5. Push contents to your GitHub repo named `CascadeEV` and connect the repo to Cloudflare Pages or GitHub Pages.

## Recommended deploy flow
- Create repo: `CascadeEV`
- Upload all files from this folder
- Connect repo to Cloudflare Pages
- Set custom domain
- Verify indexing in Google Search Console
